# Insurance EDA

EDA-Projekt (Exploratory Data Analysis) zu Versicherungskostendaten auf Basis des [Medical Cost Personal Datasets](https://www.kaggle.com/datasets/mirichoi0218/insurance) von Kaggle.

Untersucht werden Zusammenhänge zwischen Alter, BMI, Raucherstatus, Geschlecht, Region und den Versicherungskosten.

📊 **[Zum interaktiven Dashboard](https://isg-data.github.io/insurance_eda/dashboard.html)**

## Inhalt
- `dashboard.Rmd` – Quellcode des Dashboards (R / flexdashboard)
- `dashboard.html` – gerendertes Dashboard
- `insurance.csv` – Datensatz
- `EDA_in_R.ipynb` – Colab-Notebook mit der Analyse
